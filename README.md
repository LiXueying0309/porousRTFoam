# porousRTFoam
This package is designed to simulate pore-scale hydro-biogeochemical processes in porous media.

### Organization of porousRTFoam
porousRTFoam is consist of dbsFoam, dbscFoam, dissolFoam, precipFoam, and bioFoam.
